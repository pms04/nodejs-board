# Simple Node.js & Express 게시판 (PMS-Board)

이 프로젝트는 Node.js와 Express 프레임워크를 사용하여 구현한 간단한 웹 게시판입니다. MySQL 데이터베이스를 사용하며, 기본적인 사용자 인증(로그인/회원가입) 및 게시물 관리(CRUD) 기능을 제공합니다. EJS 템플릿 엔진을 사용하여 뷰를 렌더링합니다.

## 🚀 주요 기능

1.  **사용자 인증**: 이메일 기반 로그인, 비밀번호 해싱(bcryptjs)을 통한 안전한 회원가입, 세션 관리.
2.  **로그인 강제**: 로그인 사용자만 게시판 접근 허용 (`/board` 경로).
3.  **게시물 관리 (CRUD)**: 목록 보기, 검색, 상세 보기, 글쓰기, 수정, 삭제.
4.  **조회수 증가**: 상세 페이지 접속 시 조회수 자동 증가.
5.  **경로 분리**: 인증 라우터(`/`)와 게시판 라우터(`/board`) 분리.
